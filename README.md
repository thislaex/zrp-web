# Z-Roleplay Web Sitesi (Anasayfa)

Z-Roleplay, Türkiye'nin apokaliptik senaryolu SA-MP roleplay sunucusunun resmi web sitesidir. Bu proje, Z-Roleplay ekibinin sunucu tanıtımı ve topluluk bilgilendirmesi için geliştirilmiştir.

## Geliştirme Ekibi

Bu proje Z-Roleplay ekibindeyken şu geliştiriciler tarafından hazırlanmıştır:
- [thislaex](https://github.com/thislaex)
- [ebuword](https://github.com/ebuword)

## Proje Hakkında

Web sitesi, zombi istilası altındaki post-apokaliptik bir dünyada geçen SA-MP roleplay sunucusu için tasarlanmıştır. Oyuncuların sunucu hakkında bilgi alması, Discord sunucusuna katılması ve oyuna bağlanması için gerekli bilgileri içerir.

## Teknolojiler

- HTML5
- TailwindCSS
- JavaScript
- Font Awesome
- Tippy.js

## Kurulum

Projeyi yerel ortamda çalıştırmak için herhangi bir web sunucusu kullanabilirsiniz. Dosyaları indirip direkt olarak tarayıcıda açmanız yeterlidir.
